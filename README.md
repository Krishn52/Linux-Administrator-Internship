# Kali Linux VM Setup

## Host Specifications
- OS: Windows 11
- CPU: Intel Core i5
- RAM: 8 GB
- Storage: 512 GB SSD

## VM Configuration
- VM Name: Kali Linux
- RAM: 4096 MB
- CPU: 2 Cores
- Disk: 30 GB
- Network: NAT

## Installation Steps

1. Download VirtualBox.
2. Download Kali Linux ISO.
3. Create a new virtual machine.
4. Allocate RAM and storage.
5. Mount the ISO.
6. Install Kali Linux.
7. Login to Kali.
8. Update the system:
   sudo apt update
   sudo apt upgrade -y
